# Mini Mario Browser Game

A simple Mario-style platform game built with HTML, CSS, and JavaScript Canvas.

## Live Demo
https://akshathan80-max.github.io/mario-mini-js/

## Repository
https://github.com/akshathan80-max/mario-mini-js

## Controls
- Left Arrow: Move left
- Right Arrow: Move right
- Space / Up Arrow: Jump
- R: Restart

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Canvas API
